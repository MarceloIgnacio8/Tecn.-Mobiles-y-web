# Aplicación de Inicio de Sesión en Flutter

Permite a los usuarios ingresar su correo electrónico y contraseña para iniciar sesión.

## Funcionalidades

- Interfaz de usuario.
- Campos de entrada para el correo electrónico y la contraseña.
- Botón de inicio de sesión que redirige a la página de inicio con un mensaje de sesión iniciada.
- Enlaces para recordar los datos del usuario y recuperar la contraseña.
