# Actividad 3



## Descripción

Esta aplicación Flutter utiliza el paquete `carousel_slider` para crear un carrusel de tarjetas de Pokémon. Cada tarjeta incluye el nombre del Pokémon, su elemento, el número en la Pokedex y una imagen asociada. La aplicación proporciona una experiencia interactiva para explorar los Pokémon iniciales.

## Estructura del Proyecto

- **lib/mi_widget.dart**: Contiene la implementación principal de la interfaz de usuario.
- **assets/**: Carpeta que debe contener las imágenes de los Pokémon.

## Requisitos Previos

Tener instalados Flutter y Dart en tu sistema antes de ejecutar la aplicación.

## Cómo Ejecutar la Aplicación

1. **Navega a la carpeta del proyecto:**
   
    ```bash
    cd ~/Desktop/actividad3
    ```

2. **Abre Visual Studio Code en la carpeta del proyecto:**

    ```bash
    code .
    ```

3. **Asegúrate de tener las imágenes de los Pokémon:**
   
    Coloca las imágenes de los Pokémon en la carpeta `assets` dentro de la carpeta `actividad3`.

4. **Obtén las dependencias del proyecto:**

    ```bash
    flutter pub get
    ```

5. **Ejecuta la aplicación:**

    ```bash
    flutter run
    ```

La aplicación debería lanzarse.
